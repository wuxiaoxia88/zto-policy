# 更新日志

## v0.6.0 - 2026-05-09

### Added

- 新增管区视角，可在顶部与网点视角切换。
- 管区视角隐藏客户敏感字段和网点内部毛利，只展示公开政策、档位、返利变化和处罚风险。
- 新增政策宣讲首页，面向会议投屏和微信群沟通。
- 新增公开典型案例卡片，用于解释补足指标、跳档成长和规模普惠。
- 新增宣讲话术区和分享版摘要。
- 管区视角下“导出数据”只导出公开宣讲总览和公开案例，不包含客户敏感数据。

### Verified

- `pnpm test` 通过，7 个测试文件、20 个测试用例。
- `pnpm lint` 通过。
- `pnpm build` 通过。

### Notes

- 当前版本完成管区宣讲 MVP；后续可继续补会议横图模板、手机分享长图和管区批量案例库。

## v0.4.0 - 2026-05-08

### Added

- 网点视角 MVP 接入真实计算引擎，核心结果不再使用静态样例数据。
- 新增网点基础参数手工输入：月份、政策有效票量、业务量票量、业务量指标、去年同期业务量、分层基数。
- 新增日均票量滑块，拖动后自动刷新返利、跳档提示、单票返点和处罚结果。
- 新增多客户叠加测算表，支持启用、停用、追加、删除客户，并可调整件量、报价、成本、退件率。
- 新增 JSON 参数导入入口，可导入 `siteInputs` 和 `customers`。
- 新增工作台保存图片能力，便于沟通分享。
- 新增动态返利趋势曲线和增量影响拆解图。
- 决策建议面板接入真实增量拆解和业务量处罚结果。

### Verified

- `pnpm test` 通过，7 个测试文件、20 个测试用例。
- `pnpm lint` 通过。
- `pnpm build` 通过。
- 页面服务端静态渲染测试覆盖中文工作台关键内容。

### Notes

- 当前阶段优先完成网点视角可用 MVP；政策模板编辑、XLSX/CSV 导入、复制客户、管区视角和更高级动态图表将在后续版本继续完善。

## v0.3.0 - 2026-05-08

### Added

- 新增 `src/core/date.ts`，支持自然月天数和手动覆盖天数。
- 新增 `src/core/rebate.ts`，支持政策档位匹配、基础返利、分层补贴、单票返利和月度返利。
- 新增 `src/core/penalty.ts`，支持业务量未达标处罚和同比负增长处罚二选一。
- 新增 `src/core/customer.ts`，支持客户基础贡献和多客户票量叠加。
- 新增 `src/core/increment.ts`，支持增量增利/增量减利拆解。
- 新增 `src/core/recommendation.ts`，输出建议做、谨慎做、不建议做、数据不足。
- 新增苏南政策模板 `src/data/policy-presets.ts`。
- 新增计算引擎说明文档 `docs/CALCULATION.md`。

### Verified

- `pnpm test` 通过，7 个测试文件、20 个测试用例。
- 覆盖 2026-05/2026-06 月份天数。
- 覆盖苏南模板 4 个返利验收用例。
- 覆盖 8000 指标、7500 实际业务量处罚案例。
- 覆盖案例二 `94.8 + 10.4 = 105.2`，结论为增量增利。
- 覆盖案例一 `167.2 - 206 = -38.8`，结论为增量减利。
- `pnpm lint`、`pnpm build`、`pnpm build:single` 均通过。

### Notes

- 默认测试环境改为轻量 `node`，避免外置卷上 jsdom worker 启动超时。
- React 页面基线测试改为服务端静态渲染断言。
