## 4. 返利计算

核心公式:

```text
当前档位 = 按政策有效日均票量匹配档位
基础返利日票量 = Qe × (1 - 基数比例)
基础返利日金额 = 基础返利日票量 × 基础补贴
分层补贴日票量 = max(Qe - 分层基数, 0)
分层补贴日金额 = 分层补贴日票量 × 分层补贴
单日总返利 = 基础返利日金额 + 分层补贴日金额
月度总返利 = 单日总返利 × 当月天数
```

苏南模板验收:

| 输入 | 预期 |
|---|---|
| Qe=1500, B=1000 | 单日返利 1050 元 |
| Qe=2500, B=1000 | 单日返利 2662.5 元 |
| Qe=8000, B=11000 | 分层补贴 0 |
| Qe=35000, B=11000 | 单日返利 44880 元 |

## 5. 业务量处罚

核心规则:

```text
缺口 = max(业务量指标 - 实际业务量, 0)
```

如果无缺口，处罚为 0。

如果有缺口且未同比负增长:

```text
处罚 = 缺口 × 0.35 × 当月天数
```

如果有缺口且同比负增长:

```text
处罚 = 缺口 × 0.40 × 当月天数
```

两条处罚不叠加，只取命中的政策口径。

## 6. 客户贡献

客户基础贡献不直接把政策返利硬分摊到客户，先计算客户自身的收入、成本和退件收益:

```text
客户日收入 = 客户日票量 × 客户报价
客户日成本 = 客户日票量 × 客户成本
退件日收益 = 客户日票量 × 退件率 × 单件退件收益
客户基础贡献 = 客户日收入 + 退件日收益 + 其他收益 - 客户日成本
```

返利分摊、跳档收益和处罚变化由叠加方案整体计算。

## 7. 增量增利/增量减利

核心公式:

```text
最终净影响 =
  存量收益提升
  + 新增客户自身利润
  + 业务量处罚减少
  + 退件收益
  + 其他收益
```

案例二验收:

```text
94.8 + 10.4 = 105.2 元/天
结论: 增量增利
```

案例一验收:

```text
167.2 - 206 = -38.8 元/天
结论: 增量减利
```
