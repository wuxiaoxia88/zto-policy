# 原始资料说明

本项目开发时参考了部分本地原始资料，例如政策会议记录、区域标准、成本测算、利润测算案例、财务资料和品牌规范。

出于敏感信息、版权和开源合规考虑，原始资料不随开源仓库发布。开源仓库只保留经过整理后的产品文档、技术文档、UI 规范、预览图和代码。

## 本地参考资料类型

| 类型 | 用途 | 开源仓库状态 |
|---|---|---|
| 政策会议记录 | 梳理业务背景和政策口径 | 不发布 |
| 区域返利标准 | 提炼默认政策模板 | 不发布 |
| 发件成本测算 | 设计成本参数与导入结构 | 不发布 |
| 利润测算案例 | 校验增量增利/增量减利逻辑 | 不发布 |
| 财务 PDF/表格 | 理解网点经营口径 | 不发布 |
| 品牌设计规范原文 | 提取基础色彩和 UI 约束 | 不发布 |

## 开源仓库可公开内容

1. PRD 和技术规划。
2. UI 设计规范和预览图。
3. 开发文档、测试规范和使用说明。
4. 源代码、示例数据和测试用例。
5. 脱敏后的业务样例。

## 后续开发要求

后续新增示例数据时，必须使用脱敏数据，不提交客户真实名称、真实报价、内部结算数据或未授权政策原文。
